#### Current command list for Alpha 0.1

###### Starting the game
	
		in the command line, run Game.py from the relevant folder you have stored it in

		"python<version 3+> <Path to file>/Game.py"


###### Basic Commands

		These commands need to be inputted when prompted with "What you wanting to do?" in the command line

		- buy
			When prompted this will take you to the store to be able to buy the items that are on offer

		- sell
			Sell items at the store in exchange for Gold!.. Good, that.

		- leave
			Shuts the game down from/exits buy/sell mode
